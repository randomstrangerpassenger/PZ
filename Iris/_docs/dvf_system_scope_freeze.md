# DVF 시스템 동결 범위 (System Scope Freeze)

**버전**: 1.0
**상태**: ACTIVE
**날짜**: 2026-03-15

---

## 목적

DVF 메뉴 통합 단계의 포함/제외 범위를 명시하여, 범위 외 작업이 섞이지 않도록 한다.

---

## 이번 단계에 포함

1. **DVF 산출물 → 메뉴 소비자 연결**: `dvf_3_3_rendered.json` → `IrisLayer3Data.lua` → `layer3_renderer.lua`
2. **메뉴 탐색 구조 연결**: 대분류 → 소분류 → 아이템 → 설명란의 4열 구조에서 3-3 레이어 표시
3. **3-1 ~ 3-5 레이어 바인딩**: 각 계층이 자기 칸에만 연결되는 규칙 확정
4. **런타임 로딩/캐시/fallback**: `layer3_renderer.lua` fail-loud 보강
5. **메뉴 기준 통합 검증**: 데이터-메뉴 정합성, 레이어 경계(물리적+의미적), 런타임 안정성
6. **DVF freeze verdict 작성**: PASS/FAIL 판정

---

## 이번 단계에 포함되지 않음

1. **툴팁**: DVF 동결 이후 별도 축으로 개발
2. **모드 확장 시스템**: DVF 이후 단계
3. **Phase 3 재판정**: Phase 3은 이미 닫혀 있음
4. **KEEP_HOLD 재검토**: 158건은 현재 상태 유지 (3-3 비표시)
5. **KEEP_SILENT 재검토**: 832건은 현재 상태 유지 (3-3 비표시)
