**문서 유형**: 템플릿 명세서 — 기존 v1.1에 추가되는 내용

---

## 1. 공통 원칙 (v1.1 동일)

- **"행위 / 조건"이라는 단어는 설명 문장에 사용하지 않는다**
- 사용자가 실제로 쓰는 말로 풀어쓴다
- 툴팁을 보지 않아도 **어디에 쓰이는지 바로 알 수 있어야 한다**

---

## 2. Tool 계열 추가분

### Tool · Security (1-K)

```
이 아이템은 문이나 차량의 잠금을 열거나 잠그는 데 사용된다.
특정 대상에 대한 접근을 제어한다.
```

---

### Tool · Storage (1-L)

```
이 아이템은 다른 아이템을 넣어 보관할 수 있는 휴대용 용기다.
열어서 아이템을 넣거나 꺼낼 수 있다.
```

---

## 3. Furniture 계열

### Furniture · Placed (7-A)

```
이 아이템은 월드에 배치할 수 있는 이동 가능 오브젝트다.
픽업하거나 원하는 위치에 다시 놓을 수 있다.
```

---

## 4. Vehicle 계열

### Vehicle · Drivetrain (8-A)

```
이 아이템은 차량의 구동이나 주행 성능에 관여하는 부품이다.
차량에 장착하여 이동 성능을 구성한다.
```

---

### Vehicle · Body (8-B)

```
이 아이템은 차량의 외장이나 내장 구조를 구성하는 부품이다.
차량에 장착하여 차체를 구성한다.
```

---

## 5. Misc 계열

### Misc · Inert (9-A)

```
이 아이템은 게임 내에서 특별한 기능이 없는 물건이다.
사용하거나 장착할 수 없다.
```
