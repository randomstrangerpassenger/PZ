# 3-3 vs 4 Boundary Examples

> 참고: 현재 운영 기준 예시 문서는 `docs/3_3_vs_3_4_boundary_examples.md`다. 이 문서는 v2 reference note로 유지한다.

> 목적: 3-3 허용 문장과 3-4 금지 문장의 경계를 예시로 고정한다.

---

## 판정 원칙

- 허용 예시는 대표 작업 맥락을 한 줄로 요약한 3-3 문장이다.
- 금지 예시는 listification, recipe detail, action/menu text, requirement detail을 드러낸 3-4 문장이다.
- 예시는 `형태 차이`를 보여주기 위한 문서다. 실제 문면은 cluster/template 단계에서 더 다듬을 수 있다.

## 예시 쌍

| # | 허용 예시 (3-3) | 금지 예시 (3-4) | 판정 이유 |
|---|---|---|---|
| 1 | 금속 단조 작업에서 가열한 재료를 다룰 때 쓴다 | 철괴를 가열한 뒤 집어 모루에 올리고 다시 집는다 | 금지 예시는 작업 절차를 설명한다 |
| 2 | 금속 접합 작업에 쓰는 도구다 | 프로판 토치, 용접봉, 금속판을 준비해 용접한다 | 금지 예시는 requirement와 절차를 묶는다 |
| 3 | 전자 조립 작업에 들어가는 부품이다 | 무전기, 시계, 송신기, 수신기 제작에 쓰인다 | 금지 예시는 사용처 목록을 나열한다 |
| 4 | 분해나 간단한 조립 작업에 쓰는 도구다 | 우클릭 메뉴의 `Disassemble`에서 사용한다 | 금지 예시는 메뉴명과 행동명을 노출한다 |
| 5 | 통조림 개봉에 쓰는 도구다 | `Open Canned Food` 행동으로 콩 통조림, 수프 통조림을 연다 | 금지 예시는 action label과 대상 목록을 노출한다 |
| 6 | 천 재봉이나 상처 봉합 작업에 쓰는 도구다 | 실, 천 조각, 소독제를 준비해 봉합에 사용한다 | 금지 예시는 재료 목록과 절차를 섞는다 |
| 7 | 총기 개조 작업에 들어가는 부품이다 | 권총, 소총, 샷건의 특정 파츠 슬롯에 장착된다 | 금지 예시는 attach 대상 목록을 준다 |
| 8 | 탄약 제작 작업에 쓰는 도구다 | 탄피, 화약, 탄두, 프레스를 이용해 탄약을 만든다 | 금지 예시는 제작 requirement를 노출한다 |
| 9 | 조리 작업에서 재료를 익힐 때 쓴다 | 스튜, 수프, 소스, 볶음 요리 레시피에 들어간다 | 금지 예시는 recipe list다 |
| 10 | 약품 처치 작업에 쓰는 소모품이다 | 붕대보다 낫고 병원에서 더 자주 나온다 | 금지 예시는 비교와 추천이 섞인다 |
| 11 | 재료를 빻아 가공하는 작업에 쓰는 도구다 | 허브, 약재, 향신료를 갈아서 여러 레시피에 넣는다 | 금지 예시는 재료/레시피 목록이다 |
| 12 | 보관이나 휴대 작업에 쓰는 용기다 | 탄약, 의약품, 공구, 서류를 보관할 수 있다 | 금지 예시는 수용 대상 목록을 열거한다 |

## 빠른 체크리스트

- 허용 예시가 `무엇을 언제/어떤 작업에서 쓰는가`를 보여주는가
- 금지 예시가 목록, 요구 조건, 메뉴명, 절차, 비교, 추천 중 하나를 드러내는가
- 허용 예시를 더 자세히 풀면 3-4가 되어야 하고, 금지 예시는 이미 3-4 수준인지 확인한다
