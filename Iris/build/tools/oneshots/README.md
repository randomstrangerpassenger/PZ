# Iris One-Shot Build Tools

This directory contains historical analysis, investigation, and one-off mutation
scripts moved out of the `Iris/build` root during P7.

These scripts are retained for auditability and emergency reference. They are not
supported build entrypoints and should not be wired into packaging or runtime
paths without a separate review.
