# Test Mod

A Pulse mod for Project Zomboid.

## Building

1. Update uild.gradle with the correct path to Pulse.jar
2. Run: `./gradlew build`
3. Find the JAR in uild/libs/

## Installation

1. Copy the built JAR to ProjectZomboid/mods/
2. Enable the mod in-game

## Development

- Main class: $Package.TestmodMod
- Mod ID: $ModId
